[01-13] 이 책에서 자주 나오는 파이썬 용어들
#표현식(expression): 어떠한 값을 만들어 내는 간단한 코드 | 표현식 < 문장 < 프로그램
#키워드(keyword): 특별한 의미가 부여된 단어(키워드로 변수 생성X)
 *파이썬 대소문자 구분
False None True and as assert
break class continue def del elif
else except finally for from global
if import in is lambda nonlocal
not or pass raise return try
while with yield
*단어가 키워드인지 확인하는 법
  >>> import keyword
  >>> print(keyword.kwlist)

#식별자(identifier): 이름을 붙일 때 사용하는 단어 / 변수, 함수 이름 등으로 사용
-규칙:

  • 키워드 사용 X
  • 특수 문자는 언더바(_)만 허용
  • 숫자로 시작 X
  • 공백 포함 X
-스네이크 케이스(snakecase): 단어 사이에 언더 바(_)기호 붙인 경우 ex)item_list
-캐멀 케이스(camelcase): 단어 첫 글자를 대문자로 만든 경우 ex)ItemList
    유형 1: 첫 번째 글자 대문자
    유형 2: 첫 번째 글자 소문자 --- 파이썬에서 사용X

-식별자 구분 방법:
-스네이크 케이스
    괄호 O - 함수
    괄호 X - 변수
-케멀 케이스
    클래스

#주석(comment): 프로그램의 진행에 영향을 주지 않는 코드
-프로그램 설명에 사용
-주석 처리 부분 앞에 #기호
ex) >>> #간단히 출력하는 예입니다.

#연산자: 값과 값 사이에 기능을 적용할 때 사용하는 것
ex) +, -, *, /, %

#자료/리터럴(literal): 값 *상세 내용 2장 참고
ex) 1, 10, "Hello"

#출력: print()
-함수 괄호 안에 출력할 것 나열
-하나만 출력
ex)>>> print("Hello Python Programming...!")
    Hello Python Programming...!
   >>> print(52)
    52
-여러 개 출력
ex)>>> print(52, 273, "Hello")
    52 273 Hello
   >>> print("안녕하세요", "저의", "이름은")
    안녕하세요 저의 이름은
-줄바꿈: print 명령 괄호 안에 입력X 경우 줄바꿈
ex) >>> print()

    >>>
