□예제 condition05.py
#날짜/시간과 관련된 기능을 가져옵니다.
import datetime

#현재 날짜/시간을 구하고
#쉽게 사용할 수 있게 월을 변수에 저장합니다.
now = datetime.datetime.now()
month = now.month

#조건문으로 계절을 확인합니다.
if 3 <= month <= 5:
print("현재는 봄입니다.")
elif 6 <= month <= 8:
print("현재는 여름입니다.")
elif 9 <= month <=11:
print("현재는 가을입니다.")
else:
print("현재는 겨울입니다.")
 현재는 여름입니다.

#False로 변환되는 값
-if 조건문의 매개변수에 불이 아닌 다른 값이 올 때는 자동으로 불로 변환된다
 (if 조건문에 비교 연산자가 없고 어떤 변수만 놓여있을 때 대부분 이 변환 사용)
-어떤 값이 True, False로 변환되는지 알고있어야 함
  False: None, 숫자 0(0.0), 빈 컨테이너(빈 문자열, 빈 바이트열, 빈 리스트 등) // 3가지
  True: False 외 모든 것
□예제 false_value.py
print("#if 조건문에 0 넣기")
if 0:
print("0은 True로 변환됩니다")
else:
print("0은 False로 변환됩니다")
print()

print("빈 문자열은 False로 변환됩니다")

#pass 키워드
-if 조건문 사이에는 들여쓰기 4칸을 넣고 코드를 작성해야만 구문 성립
  예제) pass_keyword.py 실행 시 IndentationError 발생
  *IndentationError: 들여쓰기가 잘못되어 있다
-코드 미구현 상태 표현(개발 중이라는 의미)
1. pass 키워드 ---예제) pass_keyword01.py
if 구문 사이에 0을 넣어도 실행되지만 0의 의미를 파악하지 못할 수 있어 pass 키워드 제공

2. raise NotImplementedError: 오류 강제 발생 ---예제) pass_keyword02.py
raise 키워드 + NotImplementedError(미구현 상태 표현)

#입력을 받습니다.

#조건문 사용

#양수일 때: 아직 미구현 상태입니다.
else:
#음수일 때: 아직 미구현 상태입니다.
