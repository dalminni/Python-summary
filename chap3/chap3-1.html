[03-1] 불 자료형과 if 조건문

#Boolean(불린, 불리언): 기본적인 자료형으로 참과 거짓을 나타내는 값
-True(참), False(거짓)

#불 만들기: 비교 연산자 ==, !=, <, >, <=, >=

-숫자
ex)>>> print(10 == 100)
     False

*범위 구하기
ex)>>> x=25
     >>> print(10 < x < 30)
     True
-문자열: 사전 순서(가나다순)로 앞에 있는 것이 작은 값
ex)>>> print("가방" < "하마")
     True
     >>> print("가방" > "하마")
     False

#불 연산하기: 논리 연산자 not, and, or
-not: 아니다
 불(참과 거짓)을 반대로 전환
 단항 연산자
□예제 boolean.py
x=10
under_20=x<20
print("uder_20:", under_20)
print("not under_20:", not under_20)
 uder_20: True
not under_20: False


-and: 그리고
 피연산자 두 개 모두 참일 때 True 출력, 그 외 False
 ex)>>> print(True and True)
     True
     >>> print(True and False)
     False

-or : 또는
 피연산산자 두 개 모두 거짓일 때 False, 그 외 True
 ex)>>> print(True or False)
     True
     >>> print(False and False)
     False
