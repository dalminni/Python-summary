#자료형(data type): 자료의 형식(기능과 역할에 따라 구분된 자료의 종류)
-문자열(string): 메일, 제목, 메세지 내용 등 ex) "안녕하세요"
-숫자(number): 물건의 가격, 성적 등 ex) 52, 273
-불(boolean): 로그인 상태 등 ex) True, False

-자료 형식 확인: type() 함수
ex)
>>> print(type("hello"))
<class 'str'>
>>> print(type(273))
<class 'int'>

#문자열 만들기

-큰따옴표 ex) >>> print("hello")
-작은따옴표 ex) >>> print('hello')
-문자열 내부에 따옴표 ex) >>> print(' "hello"good morning') '"A"'
cf) 구문오류 ex) >>> print(""hello"good morning")
-이스케이프 문자(escape character): 역슬래시(\) 기호와 조합해서 사용하는 특수한 문자
\" : 큰따옴표 ex) >>> print("\"hello\"good morning")
\' : 작은따옴표 ex) >>> print('\'hello\'good morning')
\n : 줄바꿈
\t : 탭
\\ : 역슬래시(\)

#문자열 연산자
-연결 연산자 + : 문자열 연결 *문자열끼리 사용
ex)>>> print("he"+"llo")
   hello
-반복 연산자 * : 문자열 반복
ex)>>> print("hello"*3) //("3*hello")
    hellohellohello
-선택 연산자(인덱싱/indexing) [] : 문자열의 특정 위치 문자 참조

 [-1~] 뒤에서부터 선택
 유형2) 원 인덱스: 1부터 시작
